# **posmotri-v-okno**
## [Ссылка на страницу](**https://github.com/AlekseiKadochnikov/posmotri-v-okno-fd.git**)

  В проекте отрабатывались навыки расположения элементов по Flex-сетке,
Освоили работу с текстовыми полями ввода (input), с чекбоксами (checkbox).
Научились их стилизовать в различном состоянии через псевдоклассы. Работали
с кликабельностью элементов.
  В роботе использовали язык разметки для создания структуры веб-страницы и 
представления контента - html и зык стилей для описания внешнего вида документа - css

  В дальнейшем планируется улучшить работу интерактивности страницы, и адаптировать 
разметку под экраны различных устройств.
